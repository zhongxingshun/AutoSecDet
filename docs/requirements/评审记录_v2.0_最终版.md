### 关键修改点

1. **超时机制**: 单个脚本 5 分钟超时，超时自动标记 Error
2. **中断策略**: 已完成结果自动保存，当前执行用例标记 Error（中断）
3. **执行顺序**: 按分类顺序执行，同分类内按用例 ID 顺序
4. **执行日志**: 可查看 stdout/stderr，保留 30 天
5. **历史记录**: 新增 F-011，支持历史任务查询和筛选
